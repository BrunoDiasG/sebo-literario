## 📞 Perguntas Rápidas

**P: E se eu cometer um erro?**
A: Use `git log` para ver histórico. Git rastreia tudo!

**P: Como faço atualizações depois?**
A: Faça mudanças → `git push` → Netlify atualiza automaticamente

**P: Posso usar domínio próprio?**
A: Sim! No Netlify em "Domain settings"

**P: Como vejo os erros do Netlify?**
A: Em app.netlify.com → seu site → "Deploy logs"
